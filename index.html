Welcome to WONET

WONET is a simple to use java program that connects to the WORDNET project's web interface to scrape words and related information like conceptual distance and synonyms/antonyms.

The WONET project is in long term alpha and was born out of the need for an Android app I designed in college's need to access word net data but the inability of most existing Java based WordNet APIs to run smoothly on Android.

This project is maintained by thewickedaxe
